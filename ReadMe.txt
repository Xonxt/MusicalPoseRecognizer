This project uses the following technologies:


* OpenCV v3.1.0, recompiled with the OPEN_NI2 flag

* PrimeSense NiTE v2.2.0.10

* OpenNI2 v2.2

The folder, that contains the executable, needs to also contain the following folder:

  NiTE2
  │
  ├───── Data
  │      │
  │      ├───── lbsdata.idx  
  │      ├───── lbsdata.lbd
  │      ├───── lbsparam1.lbd
  │      └───── lbsparam2.lbd
  │
  ├───── FeatureExtraction.ini
  ├───── h.dat
  ├───── HandAlgorithms.ini
  └───── s.dat
